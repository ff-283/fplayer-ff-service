提示

完整链路：

① 推流端向 gateway 发 POST /api/v1/streams/start（控制面：HTTP/JSON）

② gateway 返回会话信息，编排推流 & 拉流地址

③ 推流端将媒体流推到 publishRtmp（数据面：RTMP）

④ 拉流端按需拉取 playHttpFlv / HLS / RTMP

媒体分发由内置 ZLMediaKit 内核承担，此控制台仅管理地址与状态。
